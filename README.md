# PXM
controll prox mox container and virtual machine on Flask

1. discordでログインする
2. ログインした後に自分のコンテナ一覧が表示される
3. 対象のコンテナの現在のステータスと操作ボタンが表示されており操作ボタンで電源の変更ができる

.env
```
#discord client secret
CLIENT_SECRET=

#disocrd client id
CLIENT_ID=

#flask secret key
SECRET_KEY=

#flask callback url
REDIRECT_URL=

#flask port
PORT=
```
